# Code Standards

Constants: ALL_CAPS

```javascript
const CACHE_CONTROL_HEADER = 'public, max-age=300'
```

Functions and variables: camelCase
```javascript
const createUserHandler = (req, res) => {}
```


Classes and models: PascalCase
```javascript
class AppError extends Error {}
```

Files and folders: kebab-case
```plaintext
user-handler.js
```
